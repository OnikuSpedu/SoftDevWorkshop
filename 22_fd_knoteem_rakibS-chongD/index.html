Getting started

The quickest way to get started is by using the Foundation CDN.

Add the following in the head html tag

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.7.4/dist/css/foundation.min.css" crossorigin="anonymous" /> <script src="https://cdn.jsdelivr.net/npm/foundation-sites@6.7.4/dist/js/foundation.min.js" crossorigin="anonymous" ></script>

Add the following to the body html tag

<script> $(document).foundation(); </script>
